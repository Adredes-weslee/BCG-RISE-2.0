A few things to take note of:

  1. Customer Code and Customer Name seem to have a relationship with each other. We should verify that the same Customer Code is not assigned to different Customer Names
  2. Repeat step 1 for Inventory Code and Inventory Desc
  3. When checking for duplicates, we need to take note of how the features are related to each other
    • Mulitple Sales Order No. for the same Transaction Date
    • Multiple Customer Code for the same Sales Order No.
